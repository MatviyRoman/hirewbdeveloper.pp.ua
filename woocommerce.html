WooCommerce Hooks

WooCommerce має понад 700 хуків. Я знаю де і як їх використовувати — без зайвих плагінів і правильно з точки зору безпеки і продуктивності.

WooCommerceWordPressLiqPay MonobankACF ProNova Poshta API
functions.php
// Кастомний хук після кнопки купити
add_action(
  'woocommerce_after_add_to_cart_button',
  function() {
    global $product;
    $attrs = get_field('custom_attrs',
      $product->get_id());
    if ($attrs) {
      wc_get_template(
        'custom-attrs.php',
        ['attrs' => $attrs]
      );
    }
  }
);

Питання та відповіді

Часті питання

Чим відрізняєтесь від звичайного WordPress розробника?
+

WooCommerce має власну систему хуків, рівень даних, lifecycle замовлень і платіжний API. Я спеціалізуюсь саме на WooCommerce як e-commerce платформі — знаю де і як підчепитися до процесу замовлення, як розширити продуктовий каталог і як правильно інтегрувати платіжну систему.

Чи можете інтегрувати українські платіжні системи?
+

Так. Інтегрував LiqPay, Monobank (MonoPay), PrivatBank і WayForPay у WooCommerce. Налаштовую webhook для підтвердження оплати, callback URL і повернення коштів через API.

Скільки коштує кастомна WooCommerce тема?
+

Кастомна тема по Figma: від $600. Плюс кастомні плагіни: від $200 кожен. Повний магазин з кастомним чекаутом, платіжною системою і доставкою: від $1000. Погодинно: $20/год.

Чи можете оптимізувати повільний магазин?
+

Так. Профілюю повільні запити через EXPLAIN, додаю індекси до БД, впроваджую Redis/object кешування, оптимізую зображення через WebP, зменшую кількість HTTP-запитів. Типовий результат: в 2–3 рази швидше завантаження.
